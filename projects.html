Learning Management System (LMS)

LMS allows administrator creating, managing and delivering courses. It is able to manage registered students, assigning them to courses, grading them and calculating their payments. It also has some other features, including enrol lecturers to courses, search, find and delete lecturers in their catalogues. This application used React.js for the front-end, ASP.NET Core 2.0 for the back-end, and the Entity Framework Core with a MySQL database. In addition, it is a Cloud based LMS using AMS to serve data.

Weather App

React was used to create the front-end components. Web APIs was used to get real-time weather information. This app allows searching different cities around the world. It displays both current weather condition and 5-day or 10-day forecasts.

Game PlayNine

React was used to create this simple and responsive game. When the game starts, you get a random number of stars between 1 and 9. You have the set of numbers in the bottom frame that you can use. You can select one or more numbers that would sum up to the value of the random stars. The objective is to correctly use all the numbers here in the bottom frame.
